# 常用建模操作
## 游标
- `Shift + C` 移动游标到世界原点
- `Shift + 鼠标右键` 移动游戏到鼠标位置
- `Shift + S` 游标菜单

### 编辑模式
- `e` 编辑模式下向外挤出
- `i` 编辑模式下向内基础
- `Ctrl + R` 环切
- `Ctrl + B` 倒角，比如选中一条边，倒角，就可以两边多出两条边
- `M` 合并
- `V` 断开
- `K` 切刀
- `K` 填充
- `Ctrl + E` 桥接，就是把两条边连接在一起
- `P` 分离
- `A` 全选
- `S` 缩放
- `S + Y` y方向缩放 如果选中两条边，可以使它们分开
- `S + Z` z方向缩放
- `G` 平移
- `R` 旋转
- `R + Y + 90` 沿着Y轴旋转90°
- `O` 开启/关闭衰减编辑 滑动滚轮会扩大或者减少衰减范围
- `Atl + 鼠标左键` 选中循环边

### 对象模式
- `Shift + A` 对象模式下新建

### 视图切换
- `右边数字键0` 相机视角
- `右边数字键1` 正视图
- `右边数字键3` 左视图
- `右边数字键7` 顶视图

### 其他
- `Atl + Z` 透视图 当切换到此视图的时候，可以选择被遮挡的点线面
- `Shift + 鼠标中键` 平移视图
- `Alt + G` 恢复到世界中心
