# 什么是http

超文本传输协议（HTTP）的设计目的是保证客户机与服务器之间的通信。
HTTP的工作方式客户端和服务器之前的请求-应答协议

# 两种HTTP请求方法：GET和POST

- GET - 从指定的资源请求数据
- POST - 向指定的资源提交要被处理的数据

# GET方法

请注意，查询字符串（名称/值对）是在GET请求的URL中发送的：
	
	/test/deme_from.asp?name1=value1&name2=value2

有关GET请求的其它一些注释：

- GET请求可被缓存
- GET请求保留在浏览器历史纪录中
- GET请求可悲收藏为书签
